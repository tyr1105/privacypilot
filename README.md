## ✨ 特性
- 📜 **4种文档**: 隐私政策、服务条款、Cookie政策、免责声明
- 🌍 **双语支持**: 中文和英文文档生成
- ⚖️ **合规覆盖**: GDPR、CCPA、PIPL、COPPA
- 🍪 **Cookie横幅**: 可视化创建Cookie同意横幅
- 📋 **一键复制**: 直接复制生成的文档内容
- 💾 **多格式下载**: HTML/TXT格式下载
- 🔧 **可编辑**: 生成后可自由编辑再导出
- 🆓 **完全免费**: 无需注册，无限制使用
